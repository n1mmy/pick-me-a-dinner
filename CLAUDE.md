## Project Overview

**pick-me-a-dinner** is a PostgreSQL-backed web app to help a family pick a dinner each night based on previous dinners. It tracks:
- A list of dinners per day
- A list of restaurants to order from
- A list of homecooked meals
- A dinner-adding flow with a picker for restaurant or meal, including the ability to create new restaurant entries
